# vba-mod-tools

Reusable procedures and functions for Excel automation.

## how to use

```sh
curl -o module.bas.txt https://raw.githubusercontent.com/YumaYX/vba-mod-tools/refs/heads/main/module.bas
```

## for development

### split

```sh
cargo run -- split module.bas
```

### concat

```sh
cargo run -- concat
```
